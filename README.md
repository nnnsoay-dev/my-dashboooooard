🕰️ Carpe Diem 個人復古手札 (Personal Dashboard)

這是一個結合中世紀復古美學與現代極簡風格的個人儀表板。
設計理念源自 "Carpe Diem" (及時行樂)，旨在幫助使用者優雅地管理生活、飲品紀錄與代購事業。

✨ 主要功能 (Features)

1. 📅 日程與提醒 (Planner)

艾森豪矩陣 (Eisenhower Matrix)：將待辦事項分為四個象限（緊急重要、重要不急...），提升效率。

日曆視圖：直觀的月曆介面，點擊日期即可快速安排行程。

詳細日程：支援全日事件、地點、網址連結記錄。

2. 🧋 飲品手札 (Beverage Log)

視覺化統計：自動計算年度總花費、總杯數、最愛店家與飲品。

消費趨勢圖：常駐顯示最近的飲品消費金額趨勢。

詳細紀錄：支援冰塊、甜度設定，以及「🎁 被請客」（不計入花費）與「♻️ 環保杯」標記。

飲品日曆：透過日曆檢視每個月的喝飲料頻率。

3. 🛍️ 代購管理 (Purchasing & Profit)

訂單計算機：輸入韓幣成本、匯率、售價，自動計算台幣成本與預估利潤。

營運儀表板：即時顯示淨利潤、總營收、成本結構與訂單狀態分布。

訂單狀態追蹤：從「確認中」到「訂單完成」的完整 11 階段狀態管理。

篩選與排序：可快速篩選未完成訂單，並依日期排序。

4. 💳 訂閱管理 (Subscriptions)

雙視角檢視：可切換「依服務」或「依分擔人員」查看訂閱項目。

自動加總：自動計算每位分擔人員的應付總額。

5. 💰 全球百大企業投資收益管理 (Ledger)

一般帳務：紀錄日常的「還款」與「收款」，並顯示待收/待付總額。

相機夢基金：專屬的存錢進度條，視覺化你的夢想達成率。

6. 🛠️ 小工具 (Utilities)

QR Code 產生器：輸入文字或網址即時生成二維碼。

縮網址介面：(演示功能) 模擬縮網址的操作介面。

🚀 如何使用 (How to Use)

下載：將 index.html 下載到你的電腦。

開啟：直接用瀏覽器（Chrome, Safari, Edge）打開即可使用。

資料保存：所有資料皆儲存在瀏覽器的 localStorage 中，關閉視窗後資料不會消失。

備份：使用各分頁右上角的「匯出/匯入」按鈕，定期將資料備份為 JSON 檔案。

🎨 設計風格 (Design)

配色：莫蘭迪色調 (Morandi Colors) - 米白、灰綠、霧藍、乾燥玫瑰粉。

字體：Cinzel (標題) & Lora (內文)。

元素：硬陰影 (Hard Shadows)、粗邊框、紙張紋理背景。
